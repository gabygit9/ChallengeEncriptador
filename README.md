# ChallengeEncriptador 🔐

Un proyecto de **Encriptador | Desencriptador** desarrollado como parte del desafío ONE (Oracle Next Education) en colaboración con Alura.

## 📋 Descripción

Este es un desafío de programación que implementa un encriptador y desencriptador de texto. La aplicación permite cifrar y descifrar mensajes utilizando un algoritmo específico, proporcionando una interfaz sencilla e intuitiva para los usuarios.

## 🌐 Demo en Vivo

Prueba la aplicación en vivo: [ChallengeEncriptador](https://gabygit9.github.io/ChallengeEncriptador/)

## ✨ Características

- ✅ Encriptación de textos
- ✅ Desencriptación de textos  
- ✅ Interfaz amigable e intuitiva
- ✅ Copiar texto encriptado al portapapeles
- ✅ Validación de entradas
- ✅ Diseño responsivo

## 🚀 Tecnologías Utilizadas

- **HTML5** - Estructura de la página
- **CSS3** - Estilos y diseño responsivo
- **JavaScript** - Lógica de encriptación/desencriptación

## 🔒 Algoritmo de Encriptación

La encriptación utiliza las siguientes reglas de sustitución:

- `e` → `enter`
- `i` → `imes`
- `a` → `ai`
- `o` → `ober`
- `u` → `ufat`

*Solo funciona con letras minúsculas sin acentos.*

## 💻 Cómo Usar

1. Clona el repositorio:
   ```bash
   git clone https://github.com/gabygit9/ChallengeEncriptador.git
   ```

2. Abre el archivo `index.html` en tu navegador web

3. Ingresa el texto que deseas encriptar o desencriptar

4. Haz clic en el botón correspondiente

5. ¡Listo! Tu texto ha sido procesado

## 📋 Requisitos

- Navegador web moderno (Chrome, Firefox, Safari, Edge)
- No requiere instalación adicional

## 📝 Notas Importantes

- El programa solo encripta letras minúsculas
- Los números, símbolos especiales y espacios no son encriptados
- Respeta los acentos y caracteres especiales del idioma
